# teste-motocasystems-backend
# API de Gerenciamento de Categorias e Produtos

Este é um projeto de API RESTful criado com **Laravel** e **PostgreSQL** para gerenciar categorias e produtos em um sistema. A API permite realizar operações CRUD para as entidades **Categoria** e **Produto**, incluindo a criação, leitura, atualização e exclusão de categorias e produtos.

## Tecnologias Utilizadas

- **Laravel**: Framework PHP para desenvolvimento de APIs.
- **PostgreSQL**: Banco de dados relacional.
- **Postman**: Ferramenta para testes de API.
- **PHP 8.x**: Ambiente de desenvolvimento para o projeto.

## Requisitos

- PHP 8.x ou superior
- Composer
- PostgreSQL
- Laravel 10.x ou superior
